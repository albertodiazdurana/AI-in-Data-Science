# D004: Time Budgets per Sub-Project

**Date:** 2026-03-04
**Status:** Accepted
**Context:** All sub-projects

## Decision

Allocate time budgets proportional to project complexity:
- **W2** (House Prices — Automation-Driven ML): 2 hours
- **W3** (Customer Sentiment — AI-assisted analysis): 2 hours
- **W4** (Model Explainability — SHAP/LIME): 1 hour

## Rationale

- W2 and W3 involve multiple tool categories (EDA + FE + modeling, or exploration + visualization + sentiment)
- W4 is narrower in scope (single model + two interpretation tools on a small dataset of 303 records)
- Time investment should be proportional to the assignment definition scope
- Minimal notebooks covering project requirements — not exhaustive research projects
